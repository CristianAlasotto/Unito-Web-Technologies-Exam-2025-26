# Anime Web Project

Anime Web Project is a web application developed with Node.js and Express that allows users to explore anime-related data through a server-side rendered interface and REST API endpoints.

The project includes multiple routes for managing and displaying information about anime, characters, staff members, and user profiles. It uses Handlebars as the template engine and integrates Swagger/OpenAPI documentation to describe the available API endpoints.

## Main Features

- Anime search and browsing
- Character and staff information pages
- User profile section
- REST API structure
- Server-side rendering with Handlebars
- API documentation with Swagger
- Modular Express routing
- Docker-based project setup

## Technologies Used

- Node.js
- Express.js
- JavaScript
- Handlebars
- MongoDB
- Swagger / OpenAPI
- Docker

## Project Structure

The application is organized into different Express routes, each responsible for a specific area of the project, such as anime, characters, staff, users, and profiles.

## Purpose

This project was developed as a university web programming project to demonstrate the design and implementation of a modular web application, including backend routing, API documentation, database interaction, and dynamic page rendering.
